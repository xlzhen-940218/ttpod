# TTPod (天天动听) 7.6.1 Rebuild

## 项目简介
本项目是经典音乐播放器 **天天动听 (TTPod) 7.6.1 版本** 的重新编译与修复版。旨在让这款曾经的经典应用能在现代 Android 开发环境（Android Studio）中重新焕发生命力，并修复已失效的在线服务。

## 核心进展
- **代码修复**：将 JADX 反编译出的伪代码重构为可运行的 Java 源代码，修复了大量反编译导致的逻辑错误。
- **环境适配**：全面支持最新版 Android Studio 编译，升级了 Gradle 插件及依赖库。
- **资源重构**：重新整理并还原了 `res` 资源文件，修复了部分 UI 布局及资源引用问题。
- **功能恢复与增强**：
    - **本地播放**：完美的本地音乐扫描、媒体库管理及播放体验。
    - **在线榜单**：通过内置 `FakeHttpServer` 适配，新增了排行榜数据展示（接入 QQ 音乐数据源）。
    - **动态歌词**：支持在线歌词搜索与动态显示，接入了 `lrclib.net` 等歌词库。
    - **经典音效**：完整保留并修复了经典的十段均衡器、低音增强及各类预设音效。

## 技术特性
- **编译配置**：采用 `compileSdkVersion 37`，兼容 `Android 15` 工具链。
- **接口模拟**：利用 `NanoHTTPD` 在本地构建 Mock 服务，解决了官方服务器关停导致的在线功能瘫痪。
- **现代特性**：适配了 Android 11+ 的分区存储访问权限。
